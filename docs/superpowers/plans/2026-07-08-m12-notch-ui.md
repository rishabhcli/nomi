# M12 — Notch UI, Liquid Glass & Voice Orb Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the minimal M1 surface into the full system-grade notch experience: a surface that grows from beneath the physical notch via a concave-shouldered `NotchShape`, Liquid Glass material, the blur-morph, every defined state, and the press-and-hold **voice dictation orb** (Metal shader, reactive to mic amplitude).

**Architecture:** Pure geometry/mapping/state logic lives in `MnemoOrchestrator` (unit-tested with `swift test`); the shape, Liquid Glass, Metal shader, `NSPanel`, gestures, and `SpeechAnalyzer` live in the `MnemoApp` target with manual/UI verification against [UI.md](../../../UI.md). Requires an Xcode app target for entitlements (microphone, accessibility/global hotkey).

**Tech Stack:** SwiftUI (macOS 26 Liquid Glass: `glassEffect`, `GlassEffectContainer`, `glassEffectID`), AppKit (`NSPanel`, `NSScreen`), Metal (`.metal` stitchable shader + `ShaderLibrary` + `TimelineView(.animation)`), `AVAudioEngine`, Speech (`SpeechAnalyzer`/`DictationTranscriber`/`SpeechDetector`). Builds on **M1** (`NotchState`, `NotchReducer`, `QueryService`, `NotchGeometry`) and **M4/M5/M9**.

## Global Constraints

- **Physical notch is fixed hardware** — never reshape it; the surface **grows from beneath it** with a `NotchShape` (concave top shoulders `topCornerRadius ≈ 8`, convex bottom `bottomCornerRadius` 14→28) and an **opaque-black collar** continuing the notch. Geometry **read at runtime** from `NSScreen`. (From [UI.md §2–3](../../../UI.md#2-notch-geometry-measured-at-runtime-never-hardcoded).)
- **Liquid Glass required (macOS 26)** — real APIs only; one shared `GlassEffectContainer`; glass can't sample glass. No hand-rolled blur. (From [CLAUDE.md](../../../CLAUDE.md).)
- **Voice on-device only** — `SpeechAnalyzer`; audio via `AVAudioEngine`; **must not egress**. (From [UI.md §12](../../../UI.md#12-voice-dictation--the-listening-orb).)
- **Motion tokens** from [UI.md §7 & §13](../../../UI.md#7-motion-system-curves-springs-choreography): summon `spring(0.38, 0.85)`, morph `spring(0.40, 0.86)`, grow `spring(0.32, 0.90)`; blur peak 12pt; Reduce Motion → opacity cross-fade only.
- **Voice orb must render at 120fps (ProMotion) with no dropped frames.** Bright band clamped **below the notch's bottom edge** (`Orb.maxFill = 0.80`).
- **Every terminal state is a defined output** — never a blank surface.

```swift
import CoreGraphics

/// Concave top shoulders (curve outward from the fixed notch) + convex bottom corners.
/// Verified against boring.notch's NotchShape. See UI.md §3.
public struct NotchShapeGeometry {
    public let topCornerRadius: CGFloat
    public let bottomCornerRadius: CGFloat
    public init(topCornerRadius: CGFloat = 8, bottomCornerRadius: CGFloat = 14) {
        self.topCornerRadius = topCornerRadius; self.bottomCornerRadius = bottomCornerRadius
    }
    public func path(in rect: CGRect) -> CGPath {
        let p = CGMutablePath()
        let t = topCornerRadius, b = bottomCornerRadius
        let (minX, maxX, minY, maxY) = (rect.minX, rect.maxX, rect.minY, rect.maxY)
        p.move(to: CGPoint(x: minX, y: minY))
        // concave top-left shoulder: pull down-and-in from the notch edge
        p.addQuadCurve(to: CGPoint(x: minX + t, y: minY + t), control: CGPoint(x: minX + t, y: minY))
        // down the left side to the convex bottom-left
        p.addLine(to: CGPoint(x: minX + t, y: maxY - b))
        p.addQuadCurve(to: CGPoint(x: minX + t + b, y: maxY), control: CGPoint(x: minX + t, y: maxY))
        // across the bottom to convex bottom-right
        p.addLine(to: CGPoint(x: maxX - t - b, y: maxY))
        p.addQuadCurve(to: CGPoint(x: maxX - t, y: maxY - b), control: CGPoint(x: maxX - t, y: maxY))
        // up the right side to the concave top-right shoulder
        p.addLine(to: CGPoint(x: maxX - t, y: minY + t))
        p.addQuadCurve(to: CGPoint(x: maxX, y: minY), control: CGPoint(x: maxX - t, y: minY))
        p.closeSubpath()
        return p
    }
}
```

```swift
import Foundation

public func rms(_ samples: [Float]) -> Float {
    guard !samples.isEmpty else { return 0 }
    let sum = samples.reduce(Float(0)) { $0 + $1 * $1 }
    return (sum / Float(samples.count)).squareRoot()
}

public struct EnvelopeFollower {
    let attack: Float, release: Float
    var value: Float = 0
    public init(attackMs: Double, releaseMs: Double, sampleHz: Double) {
        // one-pole coefficients per tick at sampleHz
        attack = Float(exp(-1.0 / (attackMs/1000.0 * sampleHz)))
        release = Float(exp(-1.0 / (releaseMs/1000.0 * sampleHz)))
    }
    public mutating func process(_ level: Float) -> Float {
        let coeff = level > value ? attack : release
        value = coeff * value + (1 - coeff) * level
        return min(max(value, 0), 1)
    }
}
```

```swift
import CoreGraphics
public enum OrbMapping {
    public static let maxFill: CGFloat = 0.80   // Orb.maxFill (UI.md §13)
    public static func waveHeight(amplitude a: CGFloat) -> CGFloat {
        min(maxFill, max(0, a) * maxFill)        // linear, capped at maxFill
    }
    /// Ensure the bright band never rises above the physical notch's bottom edge.
    public static func clampBelowNotch(waveTopY: CGFloat, notchBottomY: CGFloat) -> CGFloat {
        max(waveTopY, notchBottomY)
    }
}
```

- [ ] **Step 4: Metal shader + view** `Sources/MnemoApp/Shaders/Orb.metal`:

```metal
#include <metal_stdlib>
#include <SwiftUI/SwiftUI_Metal.h>
using namespace metal;

// Spectral color helper (hue → rgb)
static half3 spectral(float h) {
    float3 c = clamp(abs(fract(h + float3(0.0,0.6667,0.3333))*6.0-3.0)-1.0, 0.0, 1.0);
    return half3(c);
}

[[ stitchable ]] half4 orb(float2 pos, half4 color, float2 size, float time, float amplitude) {
    float2 uv = (pos / size) * 2.0 - 1.0;          // center to [-1,1]
    float r = length(uv);
    if (r > 1.0) return half4(0.0);                // outside the sphere
    // meniscus: 3 sine harmonics forming a flowing horizontal wave
    float w = sin(uv.x*3.0 + time*1.3)*0.25 + sin(uv.x*5.0 - time*0.7)*0.12 + sin(uv.x*9.0 + time)*0.05;
    float center = 0.0 - amplitude*0.4;            // band rises (downward-biased) with amplitude
    float band = 0.18 + amplitude*0.6;             // grows with amplitude
    float d = abs(uv.y - (center + w*amplitude));
    float intensity = smoothstep(band, 0.0, d) * (0.25 + amplitude);   // brighter with amplitude
    // chromatic aberration: offset hue sampling per amplitude
    float hue = uv.x*0.5 + time*0.15;
    half3 col = spectral(hue) * (half)intensity;
    // fresnel rim
    float rim = pow(1.0 - r, 0.4);
    col += half3(rim*0.15);
    // saturation follows amplitude (desaturate at rest)
    half lum = dot(col, half3(0.299,0.587,0.114));
    col = mix(half3(lum), col, (half)(0.2 + amplitude*0.8));
    return half4(col, (half)smoothstep(1.0, 0.98, r));
}
```
