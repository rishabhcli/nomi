# Mnemo — Implementation Plans Index

This directory holds the **granular, task-by-task** implementation plans. Each file corresponds to one milestone in [../../../PLAN.md](../../../PLAN.md) and produces working, independently-testable software. Execute in dependency order (see [PLAN.md → Appendix D](../../../PLAN.md#appendix-d--milestone-dependency-graph)).

> **For agentic workers:** each plan starts with the required-sub-skill header. Use `superpowers:subagent-driven-development` (fresh subagent per task + review) or `superpowers:executing-plans`. Steps use `- [ ]` checkboxes.

## Global rules for every plan
- **The invariant** ([CLAUDE.md §0–1](../../../CLAUDE.md)): answer path, memory, model all local; loopback only; no egress; local model only. Tests run **network-off** except the M10 egress-detection tests.
- **Deployment target macOS 26.0** (Liquid Glass). Testable logic lives in SwiftPM **library targets** (`swift test`); the app shell + AppKit/Metal live in the Xcode app target with manual/UI verification steps.
- **TDD, DRY, YAGNI, frequent commits.** Every code step shows real code; every run step shows the command and expected output.
